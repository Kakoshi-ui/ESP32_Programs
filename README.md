# ESP32_Programs
This Repository is an attempt to make a guide to programm the board ESP-WROOM-32 from Espressif Systems
The ESP32 board uses a SoC Tensilica Xtensa LX6

//*************************************************************************************************************************************************************
In this guide im using the Espressif IoT-Development-Framework (ESP-IDF) and a extension of the framework for Visual Studio Code
The programming and the flash of the board are directly using VSCode
I´ll try to include an exeample for all the modules that are integrated on the board, like the use of the GPIOS, Interrupts, Timers and the others 
pheriperals.

//*************************************************************************************************************************************************************
If any one see this repository, i hope this could help anyone to program the board, without uisng the Arduino IDE.

//*************************************************************************************************************************************************************
In the future i´ll integrate the way to programm the board using only C language from zero, without using the API´s from ESP-IDF,
only programming the board in C, controlling directly the registers.
